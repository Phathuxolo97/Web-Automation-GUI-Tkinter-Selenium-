# 🚀 Smart Web Automation GUI (Tkinter + Selenium)

Automate repetitive web tasks with a simple and user-friendly desktop application built using **Python, Tkinter, and Selenium**.

---

## 📌 Overview

This project is a **desktop GUI application** that allows users to enter personal details and automatically interact with web forms in a browser.

Instead of manually filling forms, the app:
- Opens a browser
- Navigates to a target website
- Fills in form fields automatically
- Performs actions like login or submission

## ✨ Features

- 🧑‍💻 Simple and clean GUI using Tkinter  
- 🌐 Automated browser interaction with Selenium  
- 📝 Auto-fill form fields (username, password, name, email, address)  
- 🚀 One-click automation  
- ❌ Close browser button  
- 🔧 Easy to modify for different websites  

---

## 🛠️ Tech Stack

- **Python**
- **Tkinter** – GUI
- **Selenium** – Web Automation
- **WebDriver Manager** – Automatic driver setup
- pip install selenium webdriver-manager
